# Supermarket Sales Analysis (Power BI Project)

## Project Overview
This project analyzes supermarket sales data using Power BI to uncover insights related to product performance, customer behavior, and sales trends.

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- Data Visualization

## Dataset
The dataset contains sales transactions including:
- Product line
- City
- Customer type
- Gender
- Quantity sold
- Gross income
- Payment method
- Ratings

## Key Insights

### 1. Total Sales Performance
- Total Products Sold: 1476
- Highest selling product line: Sports and Travel

### 2. Customer Analysis
- Members contributed more purchases than normal customers
- Female customers slightly higher than male customers

### 3. City Performance
- Mandalay generated the highest gross income
- Yangon and Naypyitaw also showed strong performance

### 4. Product Performance
- Home and lifestyle and Food and beverages performed strongly
- Sports and travel had the highest quantity sold

### 5. Payment Method Analysis
- Cash, Credit Card, and Ewallet were analyzed
- Cash was the most frequently used payment method

## Dashboard Features

- Interactive filters (Date, Product Line, City)
- Sales by product line
- Gross income by city
- Customer segmentation
- Profit analysis
- Rating analysis
